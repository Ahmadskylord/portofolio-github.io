Pemrograman Web

Pemrograman web adalah proses pembuatan dan pengembangan situs web serta aplikasi berbasis web. Ini melibatkan penggunaan berbagai bahasa pemrograman, teknologi, dan alat untuk merancang, membangun, dan memelihara situs web yang interaktif dan dinamis.

Belajar github

GitHub adalah platform berbasis web yang digunakan untuk pengembangan perangkat lunak dan pengelolaan kode sumber menggunakan sistem kontrol versi Git. GitHub menyediakan layanan untuk hosting repositori Git, memungkinkan pengembang untuk menyimpan, melacak, dan berkolaborasi dalam proyek perangkat lunak.

Structured Query Language (SQL)

SQL (Structured Query Language) adalah bahasa standar yang digunakan untuk mengelola dan memanipulasi basis data relasional. SQL memungkinkan pengguna untuk membuat, membaca, memperbarui, dan menghapus data dalam basis data.
